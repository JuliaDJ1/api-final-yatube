REST API для социальной сети Yatube. Позволяет публиковать посты, оставлять комментарии, подписываться на авторов и просматривать 
JWT-авторизация (access + refresh токены)
Неавторизованные пользователи — только чтение
Автор может редактировать/удалять только свой контент
Защита от подписки на самого себя (на уровне API)
Поиск подписок по username
Пагинация и фильтры
pip install -r requirements.txt
python yatube_api/manage.py migrate
python yatube_api/manage.py runserver
